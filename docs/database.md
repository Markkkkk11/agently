# Database Schema

## Enums

```sql
CREATE TYPE project_status AS ENUM ('draft', 'active', 'frozen', 'deleted');
CREATE TYPE agent_type AS ENUM ('web_developer', 'designer', 'crm_manager', 'support', 'marketer', 'seo', 'analyst');
CREATE TYPE agent_status AS ENUM ('active', 'paused');
CREATE TYPE plan_type AS ENUM ('free', 'basic', 'pro', 'ultra');
CREATE TYPE subscription_status AS ENUM ('active', 'past_due', 'frozen', 'cancelled');
CREATE TYPE message_role AS ENUM ('user', 'assistant', 'system');
```

## Tables

### users
| Column | Type | Notes |
|--------|------|-------|
| id | UUID | PK, default gen_random_uuid() |
| email | VARCHAR(255) | UNIQUE, nullable (if phone auth) |
| phone | VARCHAR(20) | UNIQUE, nullable (if email auth) |
| name | VARCHAR(255) | nullable |
| is_verified | BOOLEAN | default false |
| created_at | TIMESTAMPTZ | default now() |
| updated_at | TIMESTAMPTZ | default now() |

### projects
| Column | Type | Notes |
|--------|------|-------|
| id | UUID | PK |
| user_id | UUID | FK → users.id, ON DELETE CASCADE |
| name | VARCHAR(255) | generated by AI coordinator |
| description | TEXT | user's original idea |
| status | project_status | default 'draft' |
| created_at | TIMESTAMPTZ | |
| updated_at | TIMESTAMPTZ | |

Index: `(user_id, status)` — listing user's active projects.

### agent_templates
| Column | Type | Notes |
|--------|------|-------|
| id | UUID | PK |
| type | agent_type | UNIQUE |
| name | VARCHAR(255) | display name ("AI-разработчик сайта") |
| description | TEXT | what the agent does |
| icon | VARCHAR(50) | icon name for frontend |
| base_price | INTEGER | price in kopeks |
| capabilities | JSONB | list of features |
| system_prompt_file | VARCHAR(100) | filename in prompts/ dir |
| is_active | BOOLEAN | default true |

Pre-seeded on first migration with all agent types.

### project_agents
| Column | Type | Notes |
|--------|------|-------|
| id | UUID | PK |
| project_id | UUID | FK → projects.id, ON DELETE CASCADE |
| agent_template_id | UUID | FK → agent_templates.id |
| agent_type | agent_type | denormalized for quick access |
| status | agent_status | default 'active' |
| config | JSONB | agent-specific settings |
| created_at | TIMESTAMPTZ | |

Unique constraint: `(project_id, agent_type)` — one agent of each type per project.

### chat_messages
| Column | Type | Notes |
|--------|------|-------|
| id | UUID | PK |
| project_id | UUID | FK → projects.id, ON DELETE CASCADE |
| agent_type | agent_type | which agent this conversation is with |
| role | message_role | user, assistant, system |
| content | TEXT | message text |
| metadata | JSONB | {tokens_used, model, latency_ms, ...} |
| created_at | TIMESTAMPTZ | |

Index: `(project_id, agent_type, created_at)` — loading chat history.

### subscriptions

| current_period_end | TIMESTAMPTZ | |

| created_at | TIMESTAMPTZ | |

### usage_tracking
| Column | Type | Notes |
|--------|------|-------|
| id | UUID | PK |
| user_id | UUID | FK → users.id |
| date | DATE | |
| tokens_used | INTEGER | default 0 |
| images_generated | INTEGER | default 0 |
| requests_count | INTEGER | default 0 |

Unique constraint: `(user_id, date)` — one row per user per day.
